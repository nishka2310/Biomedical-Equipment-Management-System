# Biomedical Equipment Management System

## Project Overview

Biomedical Equipment Management System is a Django-based web application developed to manage biomedical equipment records efficiently.

This system helps users maintain, organize, search, update, and monitor equipment information through a simple dashboard interface.

---

## Features

✔ Add Equipment
✔ View Equipment List
✔ Edit Equipment Details
✔ Delete Equipment
✔ Search Equipment
✔ Dashboard Statistics
✔ Responsive UI

## Installation

Clone repository:

git clone [repository-link]

Install Django:

pip install django

Run server:

py manage.py runserver

Open browser:

http://127.0.0.1:8000/
